# 🚀 DevOps CI/CD Pipeline Project

A hands-on DevOps project built from scratch to demonstrate modern CI/CD practices using Node.js, Docker, Jenkins, Terraform, Kubernetes, and GitHub.

This project follows a real-world DevOps workflow where a simple application is progressively containerized, automated, deployed, and monitored.

---

## 📌 Project Overview

The goal of this project is to learn and implement an end-to-end CI/CD pipeline using industry-standard DevOps tools.

The project starts with a Node.js application and gradually evolves into a production-ready deployment pipeline.

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/pranay1509/devops-cicd-pipeline.git
```

### Navigate to the project

```bash
cd devops-cicd-pipeline/app
```

### Install dependencies

```bash
npm install
```

### Start the application

```bash
node index.js
```

The application will be available at:

```
http://localhost:3000
```
