# 📖 About The Project

This repository documents my journey of building a complete DevOps pipeline from scratch.

Instead of learning individual tools in isolation, this project follows a real-world software delivery workflow where each technology solves a specific problem in the deployment lifecycle.

Starting with a simple Node.js application, the project gradually evolves into a production-ready system by introducing containerization, continuous integration, infrastructure as code, container orchestration, cloud deployment, monitoring, and automated delivery.

Every phase is implemented, tested, committed, and documented individually to mirror how modern DevOps teams build and deploy applications.

---

# 🎯 Objectives

* Build a Node.js application
* Manage source code using Git & GitHub
* Containerize the application with Docker
* Orchestrate services using Docker Compose
* Publish images to Docker Hub
* Build CI pipelines with Jenkins
* Provision infrastructure using Terraform
* Deploy workloads to Kubernetes
* Host the application on Google Cloud Platform (GCP)
* Automate deployments with CI/CD
* Implement monitoring and logging
* Deliver a production-ready portfolio project

# 🐳 Running the Application

### Clone the repository

```bash
git clone https://github.com/pranay1509/devops-cicd-pipeline.git
```

### Navigate into the project

```bash
cd devops-cicd-pipeline
```

### Start the application

```bash
docker compose up --build
```

### Verify running services

```bash
docker compose ps
```

### View logs

```bash
docker compose logs
```

### Stop the application

```bash
docker compose down
```
